## ⚙️ Como rodar os projetos locais

### 1. Clone o repositório

```bash
git clone https://github.com/emersonpessoa01/growdever.git
```

### 2. Entre na pasta do desafio desejado

```bash
cd desafios-growdev
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto (via script configurado)

```bash
npm run dev
```
